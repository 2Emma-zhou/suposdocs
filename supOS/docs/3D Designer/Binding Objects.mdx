Bind real-time data to models as the trigger source of the set animations.

1. Select **APP Management** > **3D Designer** > **Project Management**. All entity templates on supOS is listed for real-time data binding.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/3dDesigner/71.jpg"/> 

2. Select a data source template, and then click **Edit** under **Operation**.
3. Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/3dDesigner/72.jpg"/> at the upper-left corner to add a model with animations configured, and then click **Add animation**.
4. Select an animiation under **Animation list**.
5. Click the box under **Expression**, and then click **Add Device attribute** to select attributes from the entity template.
6. Use the keyboard on the right to set the expression for triggering the animation, and then click **Preview** to see the effect.

<img width={750} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/3dDesigner/15.jpg"/> 

7. Click **Save**.
