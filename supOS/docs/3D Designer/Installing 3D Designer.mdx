Install 3D Designer on supOS and get authentication license before use.
## Installing 3D Designer
1. Get the compressed installation package from technical support.
2. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/factoryModeling/7.png" alt="" /> to go to the design center.
3. Select **APP Management** > **App Runtime Management**, and then click **Local Installation** at the upper-left corner.
4. Select the installation package and upload it.

<img width={450} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/3dDesigner/3.png"/>

5. After the package uploaded, click **Start** under **Operation** to start the application.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/3dDesigner/1.png"/>

## Assigning Permission
1. Select **User Center** > **User Security Management** > **User Management**, find the user you want to assign the application permission to.
2. Click **User Permission** under **Operate**.
3. Under **Design Center**, find **3D Designer** and assign permissions.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/3dDesigner/2.png"/>

4. Log in to the account you just assigned permission to, and check whether 3D Designer is listed on the menu list in design center.

## Getting License

1. Select **3D Designer** > **License**, and then download the **Machine Code**.

<img width={450} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/3dDesigner/4.png"/>

2. Send the machine code file to technical support to get the license and token.
3. Upload the license to 3D Designer, and then click **Submit**.
4. Select **App Runtime Management**, and then click **Update Configuration** under **Operation** corresponding to 3D Designer.
5. Change the token to what you got from technical support in the prompted window, and then click **Update Configuration**.

<img width={450} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/3dDesigner/5.png"/>
