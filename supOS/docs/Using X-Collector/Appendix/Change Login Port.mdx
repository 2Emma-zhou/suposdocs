When port 80 is occupied, you can change the X-Collector port in its configuration file.
1. Go to the X-Collector installation path, and then find the **nginx.conf** under **\nginx\conf**.
2. Open the file, and then change the listen port number 80 to other numbers.
![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supCollector/10.png)
3. Search for **Services** in start menu, and then find the **Supos Nginx Service** and restart it.
4. Log in to X-Collector using the new port number.
