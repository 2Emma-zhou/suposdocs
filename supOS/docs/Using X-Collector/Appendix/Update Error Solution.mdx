If X-Collector is installed on Windows Server 2012 system, prompt of **You need to install Updates corresponding to KB2919355 first** will pop up while you are installing .Net.
Follow the steps below to complete installation of system updates.
1. Download Windows Server 2012 R2 update program (KB2919442).<br/>
Address: https://www.microsoft.com/zh-cn/download/details.aspx?id=42153

2. Download Windows Server 2012 R2 update program (KB2919355) that contains a set of cumulative security updates, critical update and update.<br/>
Address: https://www.microsoft.com/en-us/download/details.aspx?id=42334

3. Install update programs according to the following order.
<ol>
<li>Windows8.1-KB2919442-x64.msu</li>
<li>clearcompressionflag.exe</li>
<li>Windows8.1-KB2919355-x64.msu</li>
<li>Windows8.1-KB2932046-x64.msu</li>
<li>Windows8.1-KB2959977-x64.msu</li>
<li>Windows8.1-KB2937592-x64.msu</li>
<li>Windows8.1-KB2938439-x64.msu</li>
<li>Windows8.1-KB2934018-x64.msu</li>
</ol>
