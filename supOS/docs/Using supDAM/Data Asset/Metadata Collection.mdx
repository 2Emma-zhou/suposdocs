Create data collection plans to collect data from specified sources for subsequent management.
1. Log in to supOS, and then select **Data Intelligence** > **Data Asset** > **Metadata collection**.
2. Click **Add collection plan**, select data source and set collection schedule.
![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supDAM/11.png)

- If the collection time is set before the plan creation time, the collection starts tomorrow at the scheduled time.
- If the collection time is set after the plan creation time, the collection starts today at the scheduled time.

3. Click **Confirm**.
:::tip
Under **Operation**, click **Collect now** to manually collect data.
:::
