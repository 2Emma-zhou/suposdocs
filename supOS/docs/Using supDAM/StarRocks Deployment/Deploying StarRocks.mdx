## Preparation
- Make sure you have the installation package of StarRocks.
- Disable swap space to eliminate the influence from wapping space.
```
echo 0 | sudo tee /proc/sys/vm/swappiness
```
- Set **overcommit_memory** to avoid out of memory issue.
```
echo 1 | sudo tee /proc/sys/vm/overcommit_memory
```
