Add data tasks with added data sources, and provided processing operator. To output processed data and map the results to other data storage.

1. log in to supOS, and then select **Data Integration Exchange** > **Task dev**.
2. Click ![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supDAM/34.png) to add a directory.
3. Click **New**, enter the task information, and then click **Confirm**.

![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supETL/24.png)

:::info
- For added tasks, you can import, export, move, view details of, edit, delete, copy and publish them.
- Make sure you created new directory before moving tasks.
:::
