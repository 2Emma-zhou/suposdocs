Upload and enable the algorithm, so the video source you added can be analyzed.
1. Select **Algorithm Manager** > **CV algorithms**, and then click **Local Install**.
2. Click **Upload** and select the algorithm package, click **OK**.
:::info
Make sure the algorithm package is packed following the instructions stated in <a href="Integrating Third-party Algorithms in X-Edge">Integrating Third-party Algorithms in X-Edge</a>.
:::
3. Click **Start** at the lower-left corner to enable the algorithm.

- Click ![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supEdge/7-1.png) to update the algorithm without changing original configurations.
- Click the video image to understand how the algorithm works.
