### Applying for Authorization on Mobile

1. Install supLink mobile on your phone, start it and enter the supLink server address, tap **Start**.
2. Enter your information under **Authorization**. <br />
<img className="image-zoom" src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/51.png" />
3. Tap **Submit**.

### Authorizing on Web

1. Log in to supLink web, and then click **Mobile Auth**.
2. Under **Authentication pending review**, click **Pass** to grant permission to the corresponding user.
![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/52.png)
3. Under **Authentication ledger**, click **Detail** to view the corresponding user information.<br />
<img className="image-zoom" src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/53.png" />

:::info
- Click **Edit** to edit the user information.
- Click **Delete** to delete the user, and then the user needs to apply for authorization again.
:::
