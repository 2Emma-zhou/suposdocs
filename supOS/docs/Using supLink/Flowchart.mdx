### Configuring on web

1. Log in to supLink web, and then click **App Manage**.
2. Under **Base App**, click **Edit** on **Flowchart**.

<img width={450} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/19.png" />

<table>
    <thead>
      <tr>
        <th>Parameter</th>
        <th>Description</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Page config</td>
        <td>Select whether to display page switch arrows and rotate icon on mobile. With them enabled, you can click arrows and icon to view the next page or rotate the page.</td>
      </tr>
      <tr>
        <td>List layout</td>
        <td>Select the layout of displaying multiple pages and folders.</td>
      </tr>
      <tr>
        <td>QR code</td>
        <td>Use supLink app to scan the code to access the application directly.</td>
      </tr>
      <tr>
        <td>Visible range</td>
        <td>Set the application visible range. Select roles to grant permissions when set <b>Visible range</b> to <b>Part visible</b> or <b>Part invisible</b>.</td>
      </tr>
      <tr>
        <td>Select group</td>
        <td>Select an app group for the application.</td>
      </tr>
      <tr>
        <td>Default open method</td>
        <td>Select the default version on which the app can be opened. You can change it to <b>Old version</b> if new version has compatibility issue.</td>
      </tr>

</table>

2. Click **Back** after saving the configuration, and then click **Setting** on **Flowchart**.
3. Click **Setting**, and then select a page from the pop-up window.

<img width={450} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/16.png" />

:::info
- Only application pages that are set to be distributed to **Mobile terminal** in supOS > **Business Designer** will be displayed.
- Self-adaptive elements include input box, drop-down list, radio, multi-selection, upload, image, cascade selection, table, data, hidden area and button. Other elements will not be displayed properly on mobile.
:::
4. (optional) Click **One click configuration** to add applications and pages under them to flowchart in batches.
:::caution
This will wipe out selected pages, and please operate with caution.
:::
5. (optional) Click **New a Folder** to add a folder and then add pages inside it.

### Viewing Effect on Mobile

1. Install supLink mobile on your phone, start it and enter the supLink server address, tap **Start**.
2. Log in with the same account as supOS.
3. Under **Base App**, tap **Flowchart**, and you can see the pages you added on web.
:::info
The **Base App** group name is set on web in **Group Manage**.
:::
4. Tap the page to see the details.<br />
