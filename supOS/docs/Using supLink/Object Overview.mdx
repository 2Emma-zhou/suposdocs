### Configuring on web
:::info
Alarm is not available for supOS-lite.
:::

1. Log in to supLink web, and then click **App Manage**.
2. Under **Base App**, click **Edit** on **Object Overview**.

<img width={450} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/46.png" />

<table>
    <thead>
      <tr>
        <th>Parameter</th>
        <th>Description</th>
      </tr>
    </thead>
    <tbody>
    <tr>
        <td>QR code</td>
        <td>Use supLink app to scan the code to access the application directly.</td>
      </tr>
      <tr>
        <td>Visible range</td>
        <td>Set the application visible range. Select roles to grant permissions when set <b>Visible range</b> to <b>Part visible</b> or <b>Part invisible</b>.</td>
      </tr>
      <tr>
        <td>Select group</td>
        <td>Select an app group for the application.</td>
      </tr>
    </tbody>
</table>

3. Click **Save**.

### Viewing Effect on Mobile

1. Install supLink mobile on your phone, start it and enter the supLink server address, tap **Start**.
2. Log in with the same account as supOS.
3. Under **Base App**, tap **Object Overview**.

:::info
- Tap **Object template** and **Object instance** to filter objects by template and instance.
- Tap **Alarm Setting** to check the object alarm settings.
- Tap **Alarm** to check the object alarms.
- Tap **Trend** to check the object trend.
:::
