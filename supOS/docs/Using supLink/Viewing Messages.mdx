Displays supOS or custom application messages sent to specific users through API to supLink mobile.
:::info
Message sending is not available for base applications on supLink.
:::


### Sending messages
1. Log in to supOS platform that is connected with supLink, and then get the AK/SK information from **System Management** > **System Confirguation** > **AK/SK credential** in the design center.
2. Log in to supLink web, select **App Manage**, and then select an app, click **Edit**.
- Enable **Subscribe message** and select a supOS application.
- Copy the app ID.
3. Enter the API information to authorization calculation code and call it to send a message to supLink mobile.
- For API information, see <a href="../../../supLink">Send Message to supLink Mobile</a>.
- For code example details, see <a href="../demo/OpenAPI demo/OpenAPI External Usage Demo">OpenAPI External Usage</a>.
- The secret key for token calculation is **1A1zP1eP5QGefi2DMPTfTL5SLmv7DivfNa** and cannot be changed.
4. Log in to the supLink app on mobile, and then tap **Message**, check the message sent from the API.
