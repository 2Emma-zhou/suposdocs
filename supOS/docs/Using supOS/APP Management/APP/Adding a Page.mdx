1. Inside the application, click **Interface Design**.
2. Select <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap1.png" /> > **Blank Page** to add a new page.

:::info Info about other icons
- <img width={20} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap6.png" />: Add folders to categorize pages.
- <img width={100} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap8.png" />: Level the selected page up/down or move it up/down.
- <img width={150} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/4.2/supos/35.png" />: Basic page operations in free layout, including preview, page settings, save current page as template, share the current page, import and export the page.
- <img width={150} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/4.2/supos/36.png" />: Basic page operations in grid layout, including preview, release the page, restore the previous step, save current page as template, share the current page, import and export the page.
:::
