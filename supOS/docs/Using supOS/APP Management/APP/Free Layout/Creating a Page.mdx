- Check tags connection.

<ol>
<ol>
<li>Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap26.png"/>, and then click <strong>Start Inspection</strong> on the pop-up window.

<img width={450} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/12.png"/>

</li>
<li>(Optional) Click <strong>Edit</strong> next to errors after inspection, replace the data source and then check again.

:::info
- **Inspection finished** is displayed when the connection is normal.
- No results are displayed when there is no bound data source.
:::

</li>
</ol>
</ol>

- Find and replace.

<ol>
<ol>
<li>Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap27.png"/>, and then enter content you want to find or replace in the pop-up window.

:::info
Data link controls are not available for text replacement.
:::

:::tip
You can also click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/14.png"/> in the textbox to select a tag or property to be replaced.
:::

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/4.2/supos/27.png"/>
</li>
<li>Set filters as needed and then click <strong>Search</strong>.</li>
<li>Set the replacement.</li>
<li>Select a replacement mode and then click <strong>Replace Preview</strong> to view the effect.</li>
<li>Select the found object, and then click <strong>Replace Selected</strong>.</li>
</ol>
</ol>

- Add datalink in batches.

<ol>
<ol>
<li>Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap28.png"/>, and then click <strong>Add Object Properties</strong> to select properties.
<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/4.2/supos/28.png"/>
</li>
<li>Set the display of selected attributes on the canvas, and then click <strong>Confirm</strong>.

:::info
When the set <strong>Integer Digits</strong> are more than that of the selected data, the data will be compensated with 0. If less, the data will be displayed as it is. For example, the data is 352.69, <strong>Integer Digits</strong> is set to 5, the data is displayed as 00352.69. If <strong>Integer Digits</strong> is set to 2, the data is displayed as 352.69.
:::

</li>
</ol>
</ol>

<ol>
