In free layout, you can create multiple blocks and place them anywhere on the page.

- Page settings: Click <strong>Settings</strong> at the upper-right corner.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/6.png"/>

<table>
<thead>
<tr>
<th>Parameter</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td>Apply on Block</td>
<td>Select whether to set the page to contain only one block. If multiple blocks, set the block size.</td>
</tr>
<tr>
<td>Layout Type</td>
<td>
<ul>
<li><strong>Flow Layout</strong>: All blocks stretch or shrink to fit the actual screen.</li>
<li><strong>Free Layout</strong>: All blocks are the set size.</li>
</ul>
</td>
</tr>
<tr>
<td>Linkage</td>
<td>Set the linkage between controls.
<ul>
<li><strong>Auto</strong>: Value changes trigger linked events automatically.
</li>
<li><strong>Manual</strong>: Click buttons to trigger linked events after values change.
</li>
</ul>
</td>
</tr>
<tr>
<td>Free Width/Free Height</td>
<td>Set the block width and height. Only available for multiple blocks in free layout.
</td>
</tr>
<tr>
<td>Mobile Adaptation</td>
<td>Set how the page adapts to mobile client screens. Only available for multiple blocks in free layout.</td>
</tr>
<tr>
<td>Background Selection</td>
<td>Set the background color.</td>
</tr>
</tbody>
</table>

:::tip how to add multiple blocks
Point to a block, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/16.png"/> on four sides of the block to add blocks.
:::
