This section introduces basic operations on the canvas in grid layout.
- Add elements: Click an element from the library, and then drag it onto the canvas.
:::info
Elements are added inside the grid displayed as red during dragging, and a layout block is added below the block displayed as red.
:::
- Delete/copy/move an element: Point to an element, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/41.png" /> in order from left to right to delete, copy and move the element.
- Select an element: You can select an element from <strong>Level</strong> list on the right, or through bread crumb at the bottom.
- Data linkage: Press ctrl and then select multiple elements to set data linkage. For details, see <a href="../../../../demo/Low%20Code%20Development/Data%20Linkage%20in%20Grid%20Layout">Data Linkage in Grid Layout</a>.

## Designing Custom Component
Custom templates can be designed and applied in application pages under **Mapping Elements**.

1. Click **Component Design** on the left side, and then add a new page.
2. Double-click the canvas, and then edit the canvas attributes on the right side.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/4.2/supos/43.png" />

<ol>
<ol>
<li>Set the component background color.</li>
<li>Under <b>Grid Settings</b>, set the grid size, grid line color and color change intervals.</li>
<li>Under <b>Custom Events</b>, click <b>Add options</b> to add a new event.

:::tip how to use the event
- Edit the event: Add the component to your application page, and then write the event script under **Custom Event**.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/4.2/supos/44.png" />

- Call the event in script: Use **custom.event alias** to call.
:::

</li>
<li>Under <b>API</b>, click <b>Add API</b>, and write the script for API function.
</li>

:::tip how to use the API
- The defined function name in your script will be the API name.
- View the API: Add the component to your application page, and then view it under **API**.
- Call the API in script: Use **component name.API.API name** to call.
:::


<li>Add events to the component under <b>Interaction</b>.</li>
</ol>
</ol>

3. Select elements from the left side to start designing.
