

1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/7.png"/> at the upper-right corner to go to the design center.
2. Select <strong>Message Management</strong> > <strong>Message Configuration</strong>, and then click the <strong>Theme Settings</strong> tab.
3. Select a theme type and then click <strong>New</strong>.
:::tip
You can add a theme type by clicking <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/28.png"/> next to the search box.
:::
4. Enter the information of the theme.

<ol>
<ol>
<li>Enter the basic information, and select the <b>Type</b> from <b>Default Type</b> and <b>To-Do Type</b>.</li>
<li>Set the <b>Receiving Range</b> by personnel, post, department or role.</li>
<li>Click <b>Choose a template</b>, select a content template you created.</li>
</ol>
</ol>

<img  width={450} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/43.png"/>

5. Click <strong>Confirm</strong>.
:::tip
You can edit or delete the template under <strong>Operation</strong>.
:::
