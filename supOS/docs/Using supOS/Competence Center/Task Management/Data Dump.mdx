

The supOS platform provides database dump to offer easy access for third_party systems to the real-time industrial data.

1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/7.png" /> at the upper-right corner to go to the design center.
2. Add a data source from <strong>Storage Resources</strong> > <strong>Resources Management</strong> to be the target database.
:::info
For details, see <a href="../Storage%20Resources/Adding Data Source">Adding Data Source</a>.
:::
3. Select **Task Management** > <strong>Live Data Dump</strong>, and then click **New Task**.
4. Enter the information of the task.

5. Under <strong>Attribute</strong>, click <strong>New</strong> to select object attributes, and then click <strong>Save</strong>.

:::info
- Up to 1000 attributes can be added.
- To add attributes to another task, we recommend to export attributes and then import to another task. You can also download a template, and enter the attribute information (<strong>Object</strong> and <strong>Attribute</strong> are required) before import.
:::

6. Click <strong>Submit</strong>.
7. Log in to the target database, and check whether the task is successfully executed.
