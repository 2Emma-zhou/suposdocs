:::
Schedule tasks for operating on applications on supOS.

1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/factoryModeling/7.png" /> to go to the design center.
2. Select <strong>Task Management</strong> > <strong>Task Scheduling</strong>.
3. Select an application from the left side, and then click **New**.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/4.2/supos/224.png" />
<table>
<thead>
<tr>
<th>Parameter</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td>Interface URL</td>
<td>The URL for executing the task.</td>
</tr>
<tr>
<td>Cron expression</td>
<td>Executon schedule of the task.</td>
</tr>
<tr>
<td>Task param</td>
<td>Necessary parameters for executing the task.</td>
</tr>
</tbody>
</table>

4. Enter task information, and then click **OK**.
5. Hover over the task, and then click **Execute the selected task now?** to start the task.
- **Edit**: Edit the task informtaion.
- **Trigger**: Edit the triggering schedule cron expressions.
- **Log**: Check the task execution logs.
- **Batch operation**: Start/stop/delete tasks in batches.
