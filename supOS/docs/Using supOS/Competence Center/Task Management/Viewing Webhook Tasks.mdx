Displays details of apps that webhook has subscribed.
1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/factoryModeling/7.png" /> to go to the design center.
2. Select <strong>Task Management</strong> > <strong>Webhook</strong>.
- When the subscription status is normal, you can check the message push logs.
- When the subscription status is stopped, you can activiate the subscription again and check the message push log.
