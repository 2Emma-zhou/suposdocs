1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/factoryModeling/7.png" /> to go to the design center.
2. Select **Object Model Management** > **Object Template** > **Relation Template**, hover over **Relation Template** and add a new template.
3. Enter the basic information of the template.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/4.2/supos/11.png" />

4. Select an **Association Mode**.
- **Binary Association**: Link 2 templates together.
- **Multivariate Association**: Link at least 3 templates together.

5. Select an **Association Type**.
- **Normal**: 
- **Combination**: 
- **Polymerization**: 

6. Under **Connection Relation**, click **Add Relationship** to add the relation between templates.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/4.2/supos/11.png" />

<ol>
<ol>
<li>Click <b>Select template</b>, and then select a template on the right list.</li>
<li>Click <b>Select template</b> to select templates to be linked.</li>
<li>Select attributes from selected templates to link them.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/4.2/supos/11.png" />
</li>
<li>Enter the relation group name and alias, and then click <b>OK</b>.</li>
</ol>
</ol>

7. (Optional) Click **Attribute**, **Services** to add attributes and services to the relation template.
