There are 5 embedded entity templates that contain different types of data collected through X-Collector as instances.

<table>
  <thead>
    <tr>
      <th>Type</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Collector Template</td>
      <td>Data collected through <b>RTD Collector</b> on X-Collector.</td>
    </tr>
    <tr>
      <td>Link Template</td>
      <td>Data connected from PI and Isys databases.</td>
    </tr>
    <tr>
      <td>Video Template</td>
      <td>Video data collected through <b>Video Collector</b> on X-Collector.</td>
    </tr>
    <tr>
      <td>History Video Template</td>
      <td>History video data collected from NVR.</td>
    </tr>
    <tr>
      <td>AE Collector Template</td>
      <td>Alarm event data collected through OPC AE.</td>
    </tr>
  </tbody>
</table>
