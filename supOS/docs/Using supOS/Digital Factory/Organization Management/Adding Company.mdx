### Adding Company

1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/factoryModeling/7.png" alt="" /> to go to the design center.
2. Select <strong>Organizational Structure</strong> > <strong>Company Management</strong>.
3. Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/factoryModeling/26.png"/>, and then enter basic information of the company.
:::tip
Select a company and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/factoryModeling/26.png"/> to create a subsidiary.
:::
4. Set system admin account for the company, and then click <strong>OK</strong>.
:::info
You can use the admin account to Log in to supOS with permissions available to the company.
:::
 
<img width={350} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/factoryModeling/19.png"/>

:::info
You cannot delete default company or edit company code and admin accounts.
:::
