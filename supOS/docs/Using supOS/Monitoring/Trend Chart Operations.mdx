### Trend Chart Configuration

### Trend Chart Settings

Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/16.png"/> to set the trend chart.

<ul>
<li>
  <strong>Trend Layout</strong>: Select information that shows on
  the trend chart.<br/>

<img width={450} height={300} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/28.png" />
  
<img width={450} height={300} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/29.png" />
  
</li>
<li>
  <strong>Numerical Coordinate Settings</strong>: Set the display
  range, font color and interval of Y axis.
  <ul>
<li>
Enable <strong>Single Y Axis</strong>, and then values of all
data sources are displayed on one Y axis. Otherwise, each data
source has corresponding Y axis.
</li>
<li>
Enable <strong>Range Adaptation</strong>, and values on Y axis
of each data source is self-adaptive and independent from each
other. Disable it and then you can set ranges for Y axis of
each data source.
</li>
<li>
Set the value display type, value font color, decimal and
value display interval of Y axis.
</li>
  </ul>
<img width={450} height={300} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/30.png" />
  
</li>
<li>
  <strong>Line Chart Settings</strong>: Set the trend chart styles.
  <ul>
<li>
<strong>Background Setting</strong>: Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/32.png"/> to switch the trend chart between dark and light mode. You can set colors of the background, chart, border, slider and font.
</li>
<li>
<strong>Grid Settings</strong>: Set the X and Y axis line type
and number of segments.
</li>
  </ul>

<img width={450} height={300} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/31.png" />
  
<img width={450} height={300} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/33.png" />
  
</li>
</ul>

### Brush Settings

Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/17.png"/> to configure the brush.

<ul>
<li>Set the trend line styles of each data sources.</li>
<li>
  Based on the set alarm rule of a data source, the data standard
  range is displayed.

:::info
- When there are multiple alarm rules, the standard
  upper/lower limit takes the lowest/highest value. You can
  change them as needed.
- In <strong>Cascading Mode</strong>, select   <strong>Exceed Limit</strong> and then data exceeds the set
  upper and lower limits is displayed in red.
:::

</li>
<li>
  Click <strong>Select Data Source</strong> to add more data
  sources. You can add up to eight data sources.
</li>
</ul>

<img width={450} height={300}   src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/34.png"/>

### Print Trend Chart

Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/18.png"/> to print the trend chart.


Select data sources and set trend line styles of each data source, and then click <strong>OK</strong> to print the trend chart of the current time period.


<img width={450} height={300}   src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/35.png"/>


### Export Trend Chart

Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/19.png"/> to export the trend chart.

<ol>
<li>
  Select data sources, and you can click <strong>Add</strong> or
  <strong>Delete</strong> to add more data or delete existing data.
</li>
<li>
  Set time period and interval between two pieces of data, and then
  click
  <strong>OK</strong> to export the trend chart.

:::caution
You can export up to 10,000 pieces of data. When exceeded, the export will fail.
:::

</li>
</ol>

<img width={450} height={300} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/36.png"/>


### Cascading Mode

Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/26.png"/> to display the trend chart in cascading mode with all trend lines separated.
<ul>
  <li>
<strong>Exceed Limit</strong>: According to the set range in <strong>Brush Settings</strong>, any data exceeds the range is
displayed in red.

<img width={450} height={300} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/37.png"/>

  </li>
  <li>
<strong>Statistics</strong>: Displays the maximum. minimum,
average and standard deviation values on the chart.

<img width={450} height={300}   src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/38.png"/>

  </li>
  <li>
<strong>Grid Lines</strong>: Displays or hides grid lines.
  </li>
</ul>
