Set applications that display on the operation center. You can either enable custom low-code and high-code apps, or directly use thrid-party system through its URL.

## Adding Menu
1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/7.png"/> at the upper-right corner to go to the design center.
2. Select <b>System Configuration</b> > <strong>Menu Configuration</strong>.
3. Select a menu on the left, and the click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/28.png"/>.
4. Enter the menu information to create a sub menu, and then click <strong>OK</strong>.

<img width={450} src="https://wordpressfreezonex.oss-cn-hangzhou.aliyuncs.com/4.2/supos/197.jpg"/>

## Configuring Menu Operation
1. Select a menu, and then click <strong>New</strong> under <strong>Operation List</strong> to add specific operations to the menu.

<img width={450} src="https://wordpressfreezonex.oss-cn-hangzhou.aliyuncs.com/4.2/supos/227.png"/>

<table>
<thead>
<tr>
<th>Parameter</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td>Operation Name</td>
<td>Set the operation name in English and Chinese by direct entering and clicking the flag icon.</td>
</tr>
<tr>
<td>Operation Code</td>
<td>The unique ID of the operation and cannot be edited.
</td>
</tr>
<tr>
<td>Restrictions</td>
<td>Select the restrictions that apply when assigning permissions of the operation. For example, select <b>Post restrictions</b>, meaning that the operation permission can be restricted by different posts.</td>
</tr>
<tr>
<td>Operation url</td>
<td>Operation list with details of each operation.</td>
</tr>
</tbody>
</table>

2. Click **New** next to **Operation url**, and then enter the request method and URL of the operation.

<img src="https://wordpressfreezonex.oss-cn-hangzhou.aliyuncs.com/4.2/supos/228.png"/>

3. Click **OK**.

:::tip to see the added menu
1. Go to **User Security Management**, and select **Operation Center**.
2. Find the menu name you added, and then assign the permission to the current user.
3. Go back to the operation center, you can see the menu name on the left side.
:::
