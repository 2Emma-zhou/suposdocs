System admin manages online users such as viewing user login list, access IP and login time. Force quit is also available when necessary.

1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/7.png" /> at the upper-right corner to go to the design center.
2. Select <strong>User Security Management</strong> > <strong>Online User</strong>.
3. Enter the username and login time, and then click <strong>Search</strong>.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/4.2/supos/212.png" />

<ul>
<ul>
<li>
<strong>Login Company</strong> shows the company the user selected during login. Aavilable when the user belongs to multiple companies.
</li>
<li><strong>Client</strong> shows the user logged in from a computer or a mobile device.
</li>
</ul>
</ul>

4. (optional) Select users, and then click <strong>Force Quit</strong> to log out the selected user accounts in batches. Directly click <strong>Force Quit</strong> under <strong>Operate</strong> to log out the corresponding user.
