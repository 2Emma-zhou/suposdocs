Requirements for installing supOS with different applications are different. Make sure to prepare hardware according to the following specifications before installation.


### supOS Alone

:::tip
1vC = 1 thread, and you can run **lscpu** command to get the number of logic CPUs.
:::

<ul>
<li>CPU: 2.2 GHz 16vCore</li>
<li>RAM: 32 GB and larger</li>
<li>
Hard drive: 2.4 TB × 3, 10000 rpm and above for a single drive, Raid5
</li>
</ul>

### Concurrency

200 ≤ Concurrent users < 2000: 

<ul>
<li>CPU: 16vCore</li>
<li>RAM: 32 GB</li>
</ul>

### i-CDS

:::info what is i-CDS
A periphral module of supOS that contains out-of-box applications you can directly install along with supOS installation.
:::

<ul>
<li>CPU: 16vCore</li>
<li>RAM: 32 GB</li>
</ul>
