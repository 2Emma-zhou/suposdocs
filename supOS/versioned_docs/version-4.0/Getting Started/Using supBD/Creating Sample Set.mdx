
Create sample sets and import data for subsequent algorithm formation. 
1. Log in to supOS, and then select **Data Intelligence** > **Data Science** > **Bigdata Analysis** > **Sample Set**.
2. Click **Add**, and then enter the basic information of the sample set.

3. Select an import method, and import data.
- Local data import

<ol>
<ol>
<li>Click <b>Local data import</b>, and then select data from local storage and import to X-BD.</li>
<li>Click <b>Table structure</b> to view the structure of the imported file and under <b>Sample data</b> to view the first 50 entries.</li>
</ol>
</ol>

![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supBD/3.png)

- OS object data sync

<ol>
<ol>
<li>Click <b>OS object data sync</b>, and then set the sync time and interval.</li>
<li>Click <b>Add</b> to select object data from supOS.</li>
<li>Click <b>Data preview</b> to view the first 50 entries.</li>
</ol>
</ol>

![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supBD/4.png)

4. Click **Submit**.
