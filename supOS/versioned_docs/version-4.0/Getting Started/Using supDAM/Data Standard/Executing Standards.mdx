Enable published standards and map data sources with standards to check whether the data source is legal.

1. Log in to supOS, and then select **Data Intelligence** > **Data Governance** > **Data standards** > **Standards execution**.
2. Select a standard set, and then click **Application mapping**.
3. Select a standard, and then click **Add mapping**.

<ol>
<ol>
<li>Select a data source for standard check.</li>
<li>Inside the data source, select fields of a data table for matching.</li>
<li>Click <b>Confirm</b>.</li>
</ol>
</ol>
