Log in to supOS, and then select **Data Intelligence** > **Data Workshop** > **Data modeling** > **Model overview**.

- Number of DIM, DWD, DWS and indicators are displayed at the top.
- The relation and process between data models and indicators are shown.
- Click sections in the process to go to corresponding menus.
