Log in to supOS, and then select **Data Intelligence** > **Data Workshop** > **O&M center** > **Data flow monitoring**.

- Data source: Displays all data sources from **Whole library migration** in X-ETL categorized by online and offline.
- Data warehouse: Classifies data flows to different layers in data warehouse.

<ul>
<ul>
<li>ODS data are from online and offline migrations in X-ETL.</li>
<li>DW tasks are from ODS, X-ETL and interfaces.</li>
</ul>
</ul>

- Data output: Indicators created based on DWS models can be output to supOS form template and services. 

:::info
Click tasks or process nodes to see corresponding details.
:::
