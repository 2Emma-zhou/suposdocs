:::info
MySQL is used as an example in this section. For details on all data source types, see <a href="Data Source Description">Data Source Description</a>.
:::

1. Log in to supOS platform, and then select **Data Integration Exchange** > **Data Connection**.
2. Click **New**, and then select **MySQL**.

![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supETL/4.png)

3. Enter the information of MySQL database, and then click **Connetcion test**.

![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supETL/5.png)

<table>
  <thead>
    <tr>
      <th>Parameter</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Affiliated business system</td>
      <td>Display category of data sources under <b>Business system</b> on <b>Data Connection</b> page. You can add custom systems as you need.</td>
    </tr>
    <tr>
      <td>Auto creation of metadata collection task</td>
      <td>Whether to generate a metadata collection task automatically in X-DAM. It is enabled and the collection is set to 1:00 by default.</td>
    </tr>
    <tr>
      <td>Connection type</td>
      <td>
      <ul>
      <li>Address/port: Enter the database server address and port, and database name for connection.</li>
      <li>JDBC: Enter JDBC connection string for connection.</li>
      </ul>
      </td>
    </tr>
  </tbody>
</table>

4. Click **Confirm** if the connection test is successful.
