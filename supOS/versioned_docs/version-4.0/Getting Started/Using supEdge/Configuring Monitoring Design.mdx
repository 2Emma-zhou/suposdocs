Design pages with different video sources and algorithms for real-time monitoring and easy alarm overview on operation center.
## Setting Real-time Video Analysis

1. Select **Design Config** > **Real-time Label** on the left panel, and then click ![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supEdge/8.png) to add a new page.
2. Design the page.
![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supEdge/7.png)

<ol>
<ol>
<li>Select whether to display the page title.</li>
<li>Set the split views.</li>
<li>Under <b>Demo screen settings</b>, select video sources and corresponding algorithms for each split view.</li>

:::tip
You can click ![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supEdge/9.png) to preview the video being analyzed by the selected algorithm.
:::


</ol>
</ol>

3. Click **Save**, and now you can view the effect on the operation center under **Real-time Label** tab.

## Setting Video Monitoring
1. Select **Design Config** > **Video Surveillance** on the left panel, and then click ![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supEdge/8.png) to add a new page.
2. Design the page.
![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supEdge/10.png)

<ol>
<ol>
<li>Select whether to display the page title.</li>
<li>Set the split views.</li>
<li>Set whether to enable video carousel to play selected sources in set time repeatedly.</li>
<li>Under <b>number screen settings</b>, select video sources and corresponding carousel duration for each split view.</li>

:::info
Up to 10 sources can be added to one split view.
:::


</ol>
</ol>

3. Click **Save**, and now you can view the effect on the operation center under **Video Surveillance** tab.


## Setting AI Display
1. Select **Design Config** > **Alarm Overview** on the left panel, and then click ![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supEdge/8.png) to add a new page.
2. Click **Add** to add linked scenes.
:::info
if you haven't added any sources and algorithms under **Real-time Label**, there will be no options.
:::

![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supEdge/11.png)

3. Click **Save**, and now you can view the effect on the operation center under **Alarm Overview** tab.

:::info
Alarms triggered by algorithms can be sent to your mobile app supLink, and you need to provide supLink information, select the video source, and set the notification style under **Notification** in the operation center.
:::
