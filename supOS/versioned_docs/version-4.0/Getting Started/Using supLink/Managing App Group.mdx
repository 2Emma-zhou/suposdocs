Create application groups for managing applications.

1. Log in to supLink web, and then click **Group Manage**.
2. Click **new**, and then enter the group name and number.
![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/15.png)
:::info
There are 4 groups by default, and we recommend you change their names to be the same as application types under **App Manage**.
:::
