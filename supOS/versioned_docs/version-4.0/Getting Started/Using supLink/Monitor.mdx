### Configuring on web

1. Log in to supLink web, and then click **App Manage**.
2. Under **Base App**, click **Edit** on **Monitor**.
![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/26.png)

<table>
    <thead>
      <tr>
        <th>Parameter</th>
        <th>Description</th>
      </tr>
    </thead>
    <tbody>
    <tr>
        <td>QR code</td>
        <td>Use supLink app to scan the code to access the application directly.</td>
      </tr>
      <tr>
        <td>Visible range</td>
        <td>Set the application visible range. Select roles to grant permissions when set <b>Visible range</b> to <b>Part visible</b> or <b>Part invisible</b>.</td>
      </tr>
      <tr>
        <td>Select group</td>
        <td>Select an app group for the application.</td>
      </tr>
      <tr>
        <td>Object list</td>
        <td>Add monitoring object to show on mobile.</td>
      </tr>
    </tbody>
</table>

3. Click **Add monitor**, and then select video sources, click **OK**.
![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/27.png)
:::info
Monitoring set by admin will show on mobile supLink apps that do not have custom object list configured.
:::
4. (optional) Click **File Folder** to add folders for video categorizing.
5. Click **Save**.

### Viewing Effect on Mobile

1. Install supLink mobile on your phone, start it and enter the supLink server address, tap **Start**.
2. Log in with the same account as supOS.
3. Under **Base App**, tap **Monitor**, and you can see the videos you added on web.
:::info
The **Base App** group name is set on web in **Group Manage**.
:::
4. Select a video source.<br />
