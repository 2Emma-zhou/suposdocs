### Configuring on web
:::info
Alarm is not available for supOS-lite.
:::

1. Log in to supLink web, and then click **App Manage**.
2. Under **Base App**, click **Edit** on **Tag Overview**.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/44.png" />

<table>
    <thead>
      <tr>
        <th>Parameter</th>
        <th>Description</th>
      </tr>
    </thead>
    <tbody>
    <tr>
        <td>QR code</td>
        <td>Use supLink app to scan the code to access the application directly.</td>
      </tr>
      <tr>
        <td>Visible range</td>
        <td>Set the application visible range. Select roles to grant permissions when set <b>Visible range</b> to <b>Part visible</b> or <b>Part invisible</b>.</td>
      </tr>
      <tr>
        <td>Select group</td>
        <td>Select an app group for the application.</td>
      </tr>
      <tr>
        <td>Tag list</td>
        <td>Select tags to be displayed on mobile.</td>
      </tr>
    </tbody>
</table>

3. Click **Add tags group**, enetr the name, and then click **OK**.
4. Click **Setting** corresponding to the tag group, and select tags in the pop-up window.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/21.png" />

5. Click **Save**.

### Viewing Effect on Mobile

1. Install supLink mobile on your phone, start it and enter the supLink server address, tap **Start**.
2. Log in with the same account as supOS.
3. Under **Base App**, tap **Tag Overview**, select a tag group and then you can see the tags.

:::info
- Tap icons at the upper-right corner, select tag category to filter displayed tags.
- Tap **Trend** of a tag to see its trend chart.
- Tap **Alarm** of a tag to see its alarm information.
:::
