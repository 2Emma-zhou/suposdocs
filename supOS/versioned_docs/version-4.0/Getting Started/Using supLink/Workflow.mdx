### Configuring on web
:::info
- Workflow is not available for supOS-lite.
- Make sure there are workflows developed on supOS.
:::

1. Log in to supLink web, and then click **App Manage**.
2. Under **Base App**, click **Edit** on **Workflow**.
![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/31.png)

<table>
    <thead>
      <tr>
        <th>Parameter</th>
        <th>Description</th>
      </tr>
    </thead>
    <tbody>
    <tr>
        <td>QR code</td>
        <td>Use supLink app to scan the code to access the application directly.</td>
      </tr>
      <tr>
        <td>Visible range</td>
        <td>Set the application visible range. Select roles to grant permissions when set <b>Visible range</b> to <b>Part visible</b> or <b>Part invisible</b>.</td>
      </tr>
      <tr>
        <td>Select group</td>
        <td>Select an app group for the application.</td>
      </tr>
      <tr>
        <td>Setting Page</td>
        <td>Add the first page for the workflow that shows on mobile when start a new workflow process.</td>
      </tr>
    </tbody>
</table>

3. Set the first page of the workflow, and then click **Save**.

### Viewing Effect on Mobile

1. Install supLink mobile on your phone, start it and enter the supLink server address, tap **Start**.
2. Log in with the same account as supOS.
3. Scroll down the workflow page from the top, scroll left and tap **New** to start a workflow.

:::info
For each task, including start a workflow and finish node tasks, a notice will be sent to supLink mobile in the **Message** tab.
:::

<ol>
<ol>
<li>Select a workflow.</li>
<li>Enter the information to start the workflow.</li>
<li>Tap <b>Submit</b>.

<img className="image-zoom" src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/32.png" />

</li>
</ol>
</ol>

4. Scroll right and select a workflow under **To-do** to finish the task.
