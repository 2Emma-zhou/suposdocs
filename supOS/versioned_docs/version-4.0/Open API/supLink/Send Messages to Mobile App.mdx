</div>
<div className='grid'>
<div className='leftCol'>Path</div>
<div className='rightCol'>

```

```

</div>
<div className='leftCol'>Header parameter</div>
<div className='rightCol'>

<table>
  <thead>
    <tr>
      <th>Parameter</th>
      <th>Required</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>agentId</td>
      <td>Optional</td>
      <td>Tenant ID. Only available for multi-tenant version</td>
    </tr>
    <tr>
      <td>Authorization</td>
      <td>Required</td>
      <td>Calculated from SDK</td>
    </tr>
    <tr>
      <td>Content-Type</td>
      <td>Required</td>
      <td>application/json</td>
    </tr>
  </tbody>
</table>

</div>
<div className='leftCol'>Body request</div>
<div className='rightCol'>

```json
{
    "appId": "", //supLink app ID
    "msg": {
        "content": "string", //required. Message content
        "link": "string", //optional. Go to the link when tapping the message
        "title": "string", //Optional. Message title
        "type": "default"  //Optional. Message UI type
    },
    "toUsers": [ //required. Message receivers
        "string"
    ]
}
```

</div>

<div className='rightCol'>

- 200: Successful.
```json

    "code": 200,
