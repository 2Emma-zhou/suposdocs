### Assigning Permissions to Subsidiary Admin

1. Create a subsidiary and department, post and staff under it.
2. Create a user under <strong>User Security Management </strong>> <strong>User Management</strong>. For details, see <a href="User Security Management">User Security Management</a>.
3. Under <strong>System Management </strong>> <strong>Menu Configuration</strong>, select an app menu and then add the subsidiary to its application when editing.
:::info
Subsidiary admin can assign APP page permissions to users in the subsidiary.
:::

<img width={350} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/factoryModeling/31.png"/>

4. Log in to supOS and go to the design center, under <strong>User Security Management</strong> > <strong>User Management</strong> or <strong>Role Management</strong>, assign permissions to specific user or role.
