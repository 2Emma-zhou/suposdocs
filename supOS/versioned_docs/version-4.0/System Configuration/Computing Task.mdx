The supOS platform provides database dump to offer easy access for third
party systems to the real-time industrial data.

1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/7.png" /> at the upper-right corner to go to the design center.
2. Add a data source from <strong>Global Configuration</strong> > <strong>Data Source Management</strong>.
:::info
For details, see <a href="Adding Data Source">Adding Data Source</a>.
:::
3. Select <strong>Computing Task</strong> > <strong>Live Data Dump</strong> to create a database dump task.
4. Click <strong>New Task</strong>, and then enter the information of the task.

5. Under <strong>Attribute</strong>, click <strong>New</strong> to select object attributes that you want to save to the database, and then click <strong>Save</strong>.

:::info
- Up to 1000 attributes can be added.
- To add attributes to another task, we recommend to export current attributes and then import to another task. You can also download a template, and enter the attribute
information (<strong>Object</strong> and <strong>Attribute</strong> are required) before import.
:::

6. Click <strong>Submit</strong>.
7. Log in to the target third party database, and check whether the new table is created.<br/>

After a period of time, the corresponding amount of data are saved to the database.
