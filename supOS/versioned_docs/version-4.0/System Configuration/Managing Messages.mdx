

1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/7.png"/> at the upper-right corner to go to the design center.
2. Select <strong>Message Management</strong> > <strong>Task Management</strong> > <strong>Task Management</strong>.
3. Set the creation time and theme, and then click <strong>Search</strong>.
4. Click <strong>Receiving Status</strong>, set filters and then click <strong>Search</strong>.<br/>
The details of message receiving, including receiver, sending
status, reading status and sending time are displayed.
