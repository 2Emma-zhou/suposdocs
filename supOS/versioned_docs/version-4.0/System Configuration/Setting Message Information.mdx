
The supOS platform provides unified message management on the notification center, which can push alarms and to-do messages generated in industrial APPs to emails and messages through APIs.


1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/7.png" /> at the upper-right corner to go to the design center.
2. Select <strong>Message Management</strong> > <strong>Task Management</strong>, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/38.png"/> at the upper-right corner.
3. Click the <strong>Method to inform</strong> tab, and then set the basic information of email and message.

<ul>
<ul>
<li>Mail
<ol>
<li>Select <strong>Mail</strong> > <strong>Basic Settings</strong>.</li>
<li>Enter the sender server IP and port, sender email and password, and set the attachment size.

:::info
**SSL** encryption is enabled by default for data security.
:::

<img width={450} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/39.png"/>

</li>
<li>Click <strong>Test Connection</strong> to test whether the email can be sent.</li>
<li>Click the <strong>Default Template</strong> tab, and then click <strong>New</strong>.</li>
<li>Enter the template name and content, and then click <strong>Confirm</strong>.</li>
</ol>
</li>
<li>
Message
<p>
Set the template of massages, whether to enable alarm and
bubble notifications, and then click <strong>Confirm</strong>.
</p>
<ul>
<li>Alarm notification means that the number of unread messages will be displayed on the homepage of the operation center.
</li>
<li> Bubble message refers to the pop-up window triggered by a new message at the lower-right corner.
</li>
</ul>

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/40.png"/>

</ul>
