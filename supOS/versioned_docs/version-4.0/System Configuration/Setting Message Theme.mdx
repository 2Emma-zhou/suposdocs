

1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/7.png"/> at the upper-right corner to go to the design center.
2. Select <strong>Message Management</strong> > <strong>Task Management</strong>, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/38.png"/> at the upper-right corner.
3. Click the <strong>Theme Settings</strong> tab, select a theme type and then click <strong>New</strong>.
:::tip
You can add a theme type by clicking <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/28.png"/> next to the search box.
:::
4. Enter the information of the theme, and then click <strong>Confirm</strong>.

:::info
The <strong>Default Type</strong> is used as an example.
:::

:::tip
You can edit or delete the template under <strong>Operation</strong>.
:::
