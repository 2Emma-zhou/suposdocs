Set the whitelist and blacklist to control platform access. IPs on the blacklist cannot access the supOS platform.

1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/7.png"/> at the upper-right corner to go to the design center.
2. Select <strong>User Security Management</strong> > <strong>Whitelist &amp; Blacklist</strong>, and then click the <strong>Whitelist</strong> tab.
3. Click <strong>Add</strong>, and then enter the IP.

:::info
Wildcard characters are available.
:::

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/34.png"/>


:::info
- If the first IP added to the list is not the current one, the platform reminds you to add the current IP to the whitelist.
- Click <strong>No</strong> on the pop-up reminder, and the current IP will be logged out of the supOS platform.
:::

4. Click the <strong>Blacklist</strong> tab to view, search for and add/delete IPs on the blacklist.
:::info
Fuzzy match is available when searching for IPs.
:::
