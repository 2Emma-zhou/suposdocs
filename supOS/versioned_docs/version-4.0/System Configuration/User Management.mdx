achieve easy management.

2. Select <strong>User Security Management</strong> > <strong>User Management</strong>, and then click <strong>New</strong>.
3. Select a user type and then enter the information.

### Assigning Permission

Aside from the permissions of the role, you can assign extra
permissions to the user.

<ul>
<li>Admin can only assign permissions to users in the same company.</li>
<li>Branch admin can only assign APP permissions to user after the system admin assigns APP permission to the branch.</li>
</ul>

1. Log in to supOS, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/7.png"/> at the upper-right corner to go to the design center.
2. Select <strong>User Security Management</strong> > <strong>User Management</strong>, and then click <strong>Permission</strong> under <strong>Operate</strong> of a user.
3. Select permissions for the user under <strong>Function Permission</strong>.

:::tip

<ul>
<li>
Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/26.png"/> next to <strong>Sync to user</strong>, you can select users in the company and assign the selected permissions to them. Up to 10 users and 50 permission can be assigned at the same time.
</li>
<li>Permissions are compatible with the ADP platform operations according to configurations including post and department restrictions, specified post, department and personnel, and handler.
</li>
</ul>
:::

4. Click <strong>Assign Permission</strong> at the upper-right corner.
:::tip
You can select an assigned permission, and drag it to the trash bin at the upper-right corner to delete it.
:::

5. Click the <strong>Data Group</strong> tab.

<ul id="assigning_permission__ul_ky5_tjf_ytb">
<li>
<strong>Resource Controlled</strong>: Assign data permissions based on the selected data group. It is the default option for all general users.
</li>
<li><strong>Resource Uncontrolled</strong>: Assign all data permissions. It is the default option for admin.
</li>
</ul>

:::info
Permissions here are linked with those of the object model management.
:::

<img width={450} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/platform/27.png"/>

6. Select the data modeling permission, and then click <strong>Assign Permission</strong>.
