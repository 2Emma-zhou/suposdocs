### Line Chart

Line charts are often used to display continuous data changes along time. Applicable for viewing data changing trends.

:::info
Spline chart is similar to line chart.
:::

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/64.png" />

<ul >
<li>
Drill-down column: When the data source is relational data, you
can add a drill-down column field, and then by clicking a data
point on the chart, more detailed data is displayed.
</li>
<li>
Backend name: Enable it and then you can use values returned
from the backend as data name.
</li>
<li>
Style: Change line width and type, node icon, size and color.
</li>
<li>
Alarm: Set the maximum and minimum values to form an alarm
range, and two red lines are displayed on the chart for easy
recognition of data out of range.
</li>
<li>Grid: Enable it to display grids.</li>
</ul>


### Bar Chart

Often used to compare values at different times or under different conditions.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/65.png" />

<ul >
<li>
Start/end color: The color at the bottom/top of the bar.
Together they form a gradient.
</li>
<li>
Drill-down column: Add a data field to the bar chart, and then
you can click the data point and view more detailed data.
</li>
<li>Overlapped ID: Overlap data columns with the same ID.</li>
<li>
Stack: Stack data columns in ways such as no stacking, general
stacking and stacking by percentage.
</li>
<li>
Alarm: Set the maximum and minimum values to form an alarm
range, and two red lines are displayed on the chart for easy
recognition of data out of range.
</li>
<li>Bar width: Set the bar width.</li>
<li>Rounded corner: Set the radius of rounded corners.</li>
<li>Grid: Enable it to display grids.</li>
<li>
Coordinate reverse: Reverse the X and Y axis, and a bar chart
changes to a horizontal bar chart.
</li>
</ul>

### Area Chart

Emphasizes the extent to which the data changes during a period.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/66.png" />

<ul >
<li>
Start/end color: The color at the bottom/top of the area.
Together they form a gradient.
</li>
<li>Type: Select the area chart type.</li>
<li>
Stack: Stack data columns in ways such as no stacking, general
stacking and stacking by percentage.
</li>
<li>
Style: Change line width and type, node icon, size and color.
</li>
<li>
Alarm: Set the maximum and minimum values to form an alarm
range, and two red lines are displayed on the chart for easy
recognition of data out of range.
</li>
<li>
Mark max. value: Enable it to highlight the maximum value of the
area chart.
</li>
<li>
Border: Set the border width, rounded corner radius and color.
</li>
<li>
font color/background color: Set the font and background color
of the labeled maximum value.
</li>
<li>Grid: Enable it to display grids.</li>
<li>Coordinate reverse: Reverse the X and Y axis.</li>
</ul>

### Dash Chart

Dash chart binds with numeric real-time data.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/67.png" />

<ul >
<li>
Start/end color: The color at the bottom/top of the dial.
Together they form a gradient.
</li>
<li>Dial color: Set the dial background color.</li>
<li>Dial border color: Set the color of dial border.</li>
<li>Dial border width: Set the width of dial border.</li>

</li>

<li>
Start/end angle: Set angles of the dial start and end scales.

<li>Font color: Set the color pf dial scale values.</li>
<li>Dial bore: Set the dial bore.</li>

### Speedometer

Speedometer binds with numeric real-time data.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/68.png" />

<ul >
<li>Dial color: Set the dial background color.</li>
<li>Dial border color: Set the color of dial border.</li>
<li>Dial border width: Set the width of dial border.</li>
<li>
Scale split: Set the number of splits of the dial. The higher
the value, the smaller the interval between scales.
</li>

<li>
Start/end angle: Set angles of the dial start and end scales.
</li>
<li>Scale color: Set the color of the dial scales.</li>
<li>Scale length: Set the length of scale lines.</li>
<li>Font color: Set the color pf dial scale values.</li>
<li>
Lateral/vertical deviation: The horizontal and vertical position
of the dial. 50 means horizontally and vertically centered.
</li>

display different level of notifications.
</li>

Origin color/radius: Set color and radius of the origin point.
</li>
<li>
Border color/width: Set color and width of the dial border.
</li>

<li>Dial bore: Set the dial bore.</li>
<li>
Display name: Set whether to display the data source value under
the origin.
</li>

Connect scale: Set whether to connect the start and end of the
speedometer border.
</li>
</ul>

### Funnel Chart

Often used to display the data decreasing. Funnel chart
 automatically sorts the data, and for different sorting sequence,
 you can convert the funnel chart to a pyramid.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/72.png" />

<ul >
<li>Type: Select between funnel chart and pyramid.</li>
<li>
Style: Set the width and height of the funnel or pyramid neck.
</li>
<li>
Border: Set the border width and color of the funnel chart.
</li>
</ul>

### Spider Chart

Mainly used to evaluate each index of a data set. X axis
 represents index variables and Y axis stands for the distance
 between the origin to each data point, which indicates the
 evaluation of each index.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/73.png" />

<ul >
<li>Style: Set line width, type, node icon, size and color.</li>
<li>Opacity: Set the opacity of the spider chart.</li>
<li>Mesh color: Set the color of each spider web line.</li>
</ul>
