
In free layout, you can create multiple blocks and place them anywhere on the page.

1. Create a page in free layout, click <strong>Setting</strong> at the upper-right corner to change page settings.

2. Disable <strong>Apply on Block</strong>, and then set the block size.
3. Point to a block, and then click icons on top to edit it.
