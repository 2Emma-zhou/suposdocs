Data linkage is necessary for data requests between input and output controls. In this section, a general data query is used as an example.
- Input controls: Input box, time controls. radios, multi-selections and more.
- Output controls: Report, table, diagram and more.

:::info
Elements used in the example are for reference only. For details on all elements, see <a href="Element Details">Element Details</a>.
:::

<span id="automatic-data-linking"></span>

## Linking Data Automatically

Form data can be linked to multiple search boxes, and automatically display results corresponding to search conditions.

:::info
Make sure you have created a form template with multiple instances as headers of the form.
:::

1. Set <strong>Linkage</strong> to <strong>Automatic</strong> in page settings, and then start designing.
2. On the canvas, add two input boxes and a table from <strong>Form Library</strong>.
3. Select the input boxes and change their name and title under <strong>Property</strong> on the right for easy recognition.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/26.png"/>

4. Select the table, under <strong>Property</strong>, set the table styles.
5. Scroll down and click the box next to <strong>Data Source</strong>, and then select the form template you have created and instances under it.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/27.png"/>

6. Click <strong>Open</strong> next to <strong>Link Object</strong>, select search type in the pop-up window, and then link two input boxes with corresponding form instances.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/28.png"/>

7. From the drop-down list next to <strong>Header Settings</strong>, select all instances from the form template you created to be headers of the table.
8. Click <strong>Save</strong>, and then click <strong>Save</strong> and <strong>Preview</strong> at the upper-right corner to try entering keywords and searching for results.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/29.png"/>

## Linking Data Manually

To avoid form data automatically displays results corresponding to search conditions, you can add a button to control the query.

1. Set <strong>Linkage</strong> to <strong>Manual</strong> in page settings, and then start designing.
2. On the canvas, add two input boxes, a button and a table from <strong>Form Library</strong>.
3. Set properties of input boxes and table referring to <a href="#automatic-data-linking">Linking Data Automatically</a>.
4. Select the button, under <strong>Property</strong>, set its name and title.
5. Select the table, click <strong>Open</strong> next to <strong>Linkage Object</strong> under <strong>Property</strong>, select search type in the pop-up window, and then link two input boxes and the button with corresponding form instances and button.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/30.png"/>

6. From the drop-down list next to <strong>Header Settings</strong>, select all instances from the form template you created to be headers of the table.
7. Click <strong>Save</strong>, and then click <strong>Save</strong> and <strong>Preview</strong> at the upper-right corner to try entering keywords and searching for results.

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/31.png"/>
