### Creating Element Template

The business designer provides general elements and extension elements
 for secondary development. You can save your design as templates for
 future use. The saved templates are under <strong>Extension Library</strong>.

:::info
General elements contain libraries such as chart, form, controls and graphics. For details, see <a href="../Element Details">Element Details</a>.
:::

1. On the new page in free layout you created, double-click or point to a block of the page, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap9.png"/> at the upper-right corner to start page design.
2. On the left side, select an element, and then click on the canvas to place the selected element.

:::tip
Drag on the canvas to set the size of the element.
:::

3. (Optional) Add folders for your templates.

<ol>
<ol>
<li>Click the <strong>Extension Library</strong> tab, select <strong>Custom Template</strong> from the drop-down menu.</li>
<li>Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap6.png"/> on the bottom.<br/>
A new folder is created. You can point to the folder, click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap3.png"/> to set a name on the pop-up window.
</li>
</ol>
</ol>

4. Right-click the element on the canvas, and then click <strong>Save As Template</strong>.

<img width={450} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/10.png"/>

5. Set the name and location of the template.

:::tip
You can download or upload templates by clicking the corresponding
icons on the bottom of <strong>Custom Template</strong> section.
:::

- Check tags connection.

<ol>
<ol>
<li>
Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap26.png"/>, and then click <strong>Start Checking</strong> on the
pop-up window.
<img width={450} src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/12.png"/>
</li>
<li>(Optional) Click <strong>Edit</strong> next to errors after inspection, replace the data source and then check again.

:::info
- <strong>The inspection is over</strong> is displayed when the connection is normal.
- No results are displayed when there is no bound data source.
:::

</li>
</ol>
</ol>

- Find and replace.

<ol>
<ol>
<li>Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap27.png"/>, and then enter content you want to find or replace in the pop-up window.

:::info
Data link controls are not available for text replacement.
:::

:::tip
You can also click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/14.png"/> in the textbox to select a tag or property to be replaced.
:::

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/13.png"/>
</li>
<li>Set filters as needed and then click <strong>Find</strong>.</li>
<li>Set the replacement using the same method as <strong>Find Content</strong>.</li>
<li>Click <strong>Replace Preview</strong> to view the effect, select the found object, and then click <strong>Replace Selected</strong>.</li>
</ol>
</ol>

- Add datalink controls in batches.

<ol>
<ol>
<li>Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap28.png"/>, and then click <strong>Add Object Properties</strong> to select properties.
<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/17.png"/>
</li>
<li>Set the display of selected properties on the canvas, and then click <strong>Confirm</strong>.

:::info
When the set <strong>Integer Digits</strong> are more than that of the selected data, the data will be compensated with 0. If less, the data will be displayed as it is. For example, the data is 352.69, <strong>Integer Digits</strong> is set to 5, the data is displayed as 00352.69. If <strong>Integer Digits</strong> is set to 2, the data is displayed as 352.69.
:::

</li>
</ol>
</ol>

- Find and replace data sources.

<ol>
<ol>
<li>
Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap29.png"/>, enter keywords, and then click <strong>Search</strong>.

:::info
Select a result, the corresponding element is highlighted on the canvas.
:::

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/15.png"/>
</li>
<li>Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/18.png"/> next to the data source, and then select another data source
to replace.

:::info
You can also directly enter a data source name following the naming rules similar to `system:Template_550:Instance_551:system:Property_554`.
:::

</li>
<li>
Click <strong>Confirm</strong>.
</li>
</ol>
</ol>
