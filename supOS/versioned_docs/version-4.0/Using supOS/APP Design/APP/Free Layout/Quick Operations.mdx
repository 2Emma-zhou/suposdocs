This section introduces some quick operations for designing a page.

<ul>
  <li>
    Click and drag on an empty space to move the canvas, and click and
    drag an element to move the element.
  </li>
  <li>Scroll the wheel to zoom in or out the canvas.</li>
  <li>
    Press ctrl and draw on the canvas to select elements inside the drawn
    area.
  </li>
  <li>
    Press ctrl and select multiple elements, and then you can configure
    properties of the same type elements.
  </li>
  <li>
    Select an element. move to the element edge, and then click{" "}
    <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/24.png"/> to rotate it.
  </li>
  <li>
    Double-click an element to open and set the <b>Dynamic Configuration</b>.
  </li>
</ul>
<p>Right-click an element, you can perform simple operations on it.</p>
