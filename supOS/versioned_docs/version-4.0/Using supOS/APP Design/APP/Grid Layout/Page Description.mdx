This section describes general pages you might see during application design in grid layout.
:::info difference from free layout
Elements can only be placed inside grids on the canvas.
:::

1. Click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap1.png"/> to add a page in grid layout, double-click or point to the page, and then click <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/Snap9.png"/> at the upper-right corner to start page design.
