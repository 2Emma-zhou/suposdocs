This section describes properties of the canvas, block and grid layout
        to offer better understanding during designing.
- Canvas property

<img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/34.png"/>

<ul>
<ul>
  <li>
    <strong>Adapt To The Content</strong>: Canvas
    size adjusts according to content and element size.
  </li>
  <li>
    <strong>Fixed Width And Height</strong>: Select
    it and set the width and height of the canvas. Unit can be
    percentage or pixel.
  </li>
  <li>
    <strong>Number of Columns</strong>: Set the
    number of columns on the canvas. Click{" "}
    <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/appDesign/35.png"/> to add or delete columns.

:::info
Select units of columns from <strong>Proportion</strong> and <strong>Pixel</strong>, the supOS platform arranges the width of each column based on units and number of columns.
:::

</li>
</ul>
</ul>

- Canvas interact<br/>
Select <strong>Interact</strong> next to canvas property, and then set its interact events. For details, see <strong>Interact</strong> in <a href="../Free Layout/Page Description"> Page Description</a>.

      </td>
      <td>
        The block is at the upper-left corner of the column and adapts

    <tr>
      <td>
        Absolute Position
      </td>
      <td>
        <ul>
<li>
  The block takes up a whole column without setting specific
  width and height.
</li>

  With set width and height smaller than the column, you can
  select the block from <strong>Level</strong> list, point to it

      </td>

        scroll bars are displayed.
      </td>
    </tr>
    <tr>
      <td>
        Fit Window
      </td>
      <td>
        Block size is the same as the left area of its parent block.
      </td>
    </tr>
    <tr>
      <td>
        Adapt To The Content
      </td>
      <td>
        Block size adjusts according to content and element size. You
        need to set the width and height of elements inside the block.
      </td>
    </tr>
    <tr>
      <td>
        Display Level
      </td>
      <td>
        Set the display level of the block. The higher the level, the
        more front it is displayed.
      </td>

      <td>
        Background Color
      </td>
      <td>
        Set the block background color.
      </td>
    </tr>
    <tr>
      <td>
        Upload Icon
      </td>
      <td>
        Upload an image to be the block background image.
      </td>
    </tr>
    <tr>
      <td>
        Aspect Ratio
      </td>
      <td>
        Select the background image display.
        <ul>
<li>
  <strong>No Zoom</strong>: Displays the
  image as it is.
</li>
<li>
  <strong>Subject to Width</strong>:
  Stretches the image in equal proportion with the same
  width as the block.
</li>
<li>
  <strong>Subject to Height</strong>:
  Stretches the image in equal proportion with the same
  height as the block.
</li>
<li>
  <strong>Covered with</strong>: The
  background image covers the whole block.
</li>
        </ul>
      </td>
    </tr>

You can set rows and columns of the grid, and set the width of each column.
