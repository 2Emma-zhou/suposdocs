In the operation center of supOS platform, you can monitor the running status of field equipment to realize easier operation and maintenance.

## User Desktop

When the default homepage is set to <strong>Desktop System / Default Desktop</strong> from <img src="https://wordpressfreezonex.oss-accelerate.aliyuncs.com/monitoring/6.png" /> > <strong>System information management</strong> > <strong>Default Home Page</strong> in design center, you will see a user desktop after logging in to supOS platform.

Click <strong>Normal View</strong> on the user desktop to go to normal desktop, and for details of normal desktop interface, see <a href="../Login#operation-center">Logging in to Operation Center</a>.
