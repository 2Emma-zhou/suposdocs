- **Create object template**: Using the connected collector data or thrid-party databases to create object templates, instances and subordinate attributesm services and events that are parts of applications.
- **Assign permissions**: Assign permissions of applications and other function modules to corresponding companies and staff.
- **Monitor applications**: With the right permission, user can monitor and perform necessary operations on applications in operation center, to obtain more information during business processes.
